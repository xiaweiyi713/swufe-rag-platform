## C 模块：路由、生成与门禁

已完成：

- “专业 + 年级 + 课程/学分/培养方案”等学校事实强制走 `school_rag`。
- 证据不足时不回退普通模型猜学校事实。
- 证据门检查最低相似度、必需实体和年级版本，不再因无关高相似文本误放行。
- DeepSeek 回答必须逐句带引用；数字、课程代码必须出现在引用原文中。
- 引用格式修复失败、引用越界、数字不受支持或回答混入“证据不足”时均 fail closed。
- “大一要修什么课”改用结构化课程目录聚合第1/2学期，再绑定原表块，避免向量检索无法完整枚举课程。
- 临时 API Key 仅通过 `X-LLM-API-Key` 请求头传入，按请求创建客户端，不写入 `.env`、YAML、SQLite、URL、日志或浏览器持久化存储。
